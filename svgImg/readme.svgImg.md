# SvgImg: Animated image for Github README
## Specialization
- A simple SVG animation with a 1920x1080 background with 5 scrolling 500x500 images

## How to use
- Allow Github action "Read and Write" (Settings > Actions > General > Workflow permissions)
- Copy this folder (svgImg) into your repo's root directory
- Replace 5 500x500 images and 1920x1080 (if needed) images on your choose in assets folder, filenames must match exactly, format allows png jpg and jepg.
- Usage:
```markdown
![Image](./svgImg/output.svg)
```
